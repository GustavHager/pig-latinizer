# pig-latinizer
For the software development phd-course. An application to turn proper english into pig-latin.


To download
git clone git@github.com:GustavHager/pig-latinizer.git
cd pig-latinizer
git submodule init
git submodule update

To build:
mkdir build && cd build && cmake .. && make

Dependencies:
Google test (gtest)
